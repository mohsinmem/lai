# Leadership Adaptiveness Institute (LAI)

Establishing the Global Standard for Leadership Adaptiveness.

## Overview

The Leadership Adaptiveness Institute (LAI) is a global research body dedicated to measuring and standardizing how leaders and organizations respond to changing environments. In an age of perpetual disruption, the primary challenge facing modern leadership is not strategy or execution, but **adaptiveness** — the ability to translate environmental awareness into effective action.

This repository contains the official web platform for LAI, featuring research insights, the Five Dimensions framework, and the interactive Leadership Adaptiveness Diagnostic (LAD).

## Key Features

- **Institutional Presence**: High-authority design tailored for global research and strategy bodies.
- **Adaptive Framework**: Comprehensive documentation of the behavioral system behind leadership adaptiveness.
- **Interactive Diagnostic**: A 20-question professional assessment tool with dynamic scoring and reporting.
- **Global Index**: Transparent benchmarking of organizational and regional adaptive capacity.

## Tech Stack

- **Framework**: React 18 (Vite)
- **Styling**: Vanilla CSS (Custom Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide-React
- **Routing**: React Router 6

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mohsinmem/lai.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Development

The project uses a custom design system defined in `src/index.css` via CSS variables. All components are built to be modular and responsive.

- **`src/components`**: Shared UI elements (Navbar, Footer, etc.)
- **`src/pages`**: Main content areas and interactive tools.
- **`src/styles`**: Global styles and utility classes.
